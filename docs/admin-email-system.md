# Admin 郵件群發系統使用說明

## 功能概覽

管理員郵件群發系統允許您向系統中的使用者批量發送郵件。支援多種預設模板，並可自訂內容。

## 主要功能

### 1. 導航位置
- 登入管理員帳號後，在頂部導航欄找到「郵件群發」選項
- 路徑：`/admin/email`

### 2. 收件人選擇
- **搜尋功能**：可通過姓名或 Email 搜尋使用者
- **篩選功能**：
  - 所有使用者
  - 活躍使用者
  - 付費使用者
  - 免費使用者
- **批量選擇**：支援全選/取消全選功能
- **多選操作**：可逐個勾選需要發送的使用者

### 3. 郵件模板

系統提供 5 種預設模板：

1. **系統公告**：用於發送系統更新、新功能通知等公告
2. **功能更新**：通知用戶新功能發布和產品更新
3. **優惠活動**：推廣付費方案和特別優惠
4. **電子報**：定期發送履歷撰寫技巧和職涯建議
5. **自訂郵件**：完全自訂郵件內容

### 4. 內容編輯

#### 郵件主旨
- 所有模板都可以修改郵件主旨
- 預設模板會自動填入建議主旨

#### 郵件內容
- **預設模板**：可以自訂內容來替換預設模板內容
- **自訂模板**：必須輸入完整的郵件內容
- 支援 HTML 格式

### 5. 即時預覽功能

#### 啟用即時預覽
1. 勾選「啟用即時預覽」選項
2. 當您修改主旨或內容時，預覽會自動更新
3. 預覽會顯示在頁面底部

#### 手動預覽
1. 關閉即時預覽時，可點擊「預覽郵件」按鈕
2. 預覽會以彈出視窗顯示

### 6. 發送郵件

1. 選擇收件人（至少選擇一位）
2. 選擇郵件模板
3. 編輯主旨和內容
4. 預覽郵件內容
5. 點擊「發送郵件」按鈕
6. 確認發送數量
7. 系統會批量發送並顯示結果

## 注意事項

1. **發送限制**：系統會分批發送，每批 10 封，間隔 1 秒，避免觸發郵件服務商的頻率限制
2. **必填項目**：
   - 至少選擇一位收件人
   - 必須填寫郵件主旨
   - 自訂模板必須填寫內容
3. **發送結果**：系統會顯示成功和失敗的數量

## 最佳實踐

1. **測試發送**：建議先選擇少量使用者測試郵件內容
2. **預覽確認**：發送前務必預覽郵件內容
3. **分批發送**：大量郵件建議分批次發送
4. **內容檢查**：確保郵件內容符合規範，避免被標記為垃圾郵件

## 環境變數配置

確保以下環境變數已正確設定：
- `RESEND_API_KEY`：Resend 服務的 API 金鑰
- `FROM_EMAIL`：發送郵件的寄件人地址
