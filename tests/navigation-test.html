Enhanced Inline Text Navigation Test

Test Instructions:

This page tests the enhanced navigation behaviors:

  1. 行間 ↑ → 行首 (In-line Up → Line Start)
  2. 行間 ↓ → 行末 (In-line Down → Line End)
  3. 行首 ↑/← → 上一行末 (Line Start Up/Left → Previous Line End)
  4. 行末 ↓/→ → 下一行首 (Line End Down/Right → Next Line Start)
  5. 第一個列點的行首 ↑/← → 上一個欄位 (First Bullet Line Start → Previous Field)
  6. 最後一個列點的行末 ↓/→ → 下一個欄位 (Last Bullet Line End → Next Field)

Manual Testing Checklist:

  • ✅ Single-line: Up/Down arrows move between fields
  • ✅ Multi-line: Up/Down in middle move to line start/end
  • ✅ Multi-line: Up/Down at line boundaries move to prev/next line
  • ✅ Multi-line: Left/Right at line boundaries move to prev/next line
  • ✅ Field boundaries: Up/Down/Left/Right move between fields
  • ✅ Bullet navigation works within groups
  • ✅ Cross-field navigation respects order

How to Test:

  1. Click into any field to focus it
  2. Use arrow keys to navigate and observe behavior
  3. Test each scenario systematically
  4. Verify that navigation behaves as described in requirements
